## PIVOT
데이터를 검색할 때 **행 집합을 열로 변환**시켜 결과값을 보여준다.

## PIVOT 문법
```sql
SELECT 컬럼목록
FROM (
    SELECT 원본컬럼들
    FROM 테이블명
)
PIVOT (
    집계함수(값컬럼)
    FOR 기준컬럼 IN (
        '값1' AS 새컬럼1,
        '값2' AS 새컬럼2,
        ...
    )
);
```
- 집계함수: 보통 ```SUM, MAX, COUNT``` 등을 사용 (*필수)
- 값컬럼: 실제로 집계할 숫자나 수치 데이터
- 기준컬럼: 피벗 기준이 되는 열 (예: 날짜, 카테고리 등)
- IN절: 어떤 값들을 새로운 열로 바꿀지 지정 '값' ```AS``` 새컬럼이름 형식

## PIVOT 예제
```SQL
SELECT *
FROM (
    SELECT 품목, 날짜, 가격
    FROM 과일_판매
)
PIVOT (
    MAX(가격)
    FOR 날짜 IN (
        'Y2025_07' AS "202507",
        'Y2025_08' AS "202508",
        'Y2025_09' AS "202509"
    )
);
```
## UNPIVOT
데이터를 검색할 때 **열을 행의 집합**으로 보여준다.

## 기본 문법
```SQL
SELECT 컬럼1, 컬럼2, ...
FROM 테이블명
UNPIVOT (
    값이들어갈컬럼 FOR 기준컬럼 IN (기존의 열 목록)
);
```

## UNPIVOT 예제
```SQL
SELECT 품목, 날짜, 가격
FROM 과일_가격
UNPIVOT (
    가격 
    FOR 날짜 IN (
        Y2025_07, 
        Y2025_08, 
        Y2025_09
    )
);
```
