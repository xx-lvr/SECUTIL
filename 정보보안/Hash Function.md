## 해시 함수(Hash Function)
**해시 함수**는 임의의 길이의 입력 데이터를 고정된 길이의 출력 값으로 변환하는 함수이다.

+ ```대표 알고리즘: MD5, SHA-1, SHA-256, SHA-384, SHA-512```
## 주요 특징
+ 목적: **데이터의 무결성 확보**

+ 일방향성: 입력값을 **해시값으로 변환하는 것은 가능**하지만,
해시값으로부터 **원래의 입력값을 역추적하는 것은 불가능**하다.
+ 고정성:동일한 입력값에 대해서는 **항상 동일한 해시값**을 반환한다.
+ 고유성: 서로 다른 입력값이 같은 해시값을 가질 가능성(충돌) 은 존재하지만, 그 확률이 매우 낮도록 설계되어 있다.
+ 불변성: 입력값이 조금만 변경되어도 **해시값은 완전히 다른 값으로 변경**된다.
