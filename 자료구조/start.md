## 알고리즘이란?
어떤 문제를 해결하기 위해 나열해놓은 일련의 과정

알고리즘의 5가지 조건

- 입력(Input) → 0개 이상

- 출력(Output) → 1개 이상

- 명확성(Definiteness) → 각 단계가 애매하지 않음

- 유한성(Finiteness) → 반드시 끝남 (무한루프 X)

- 유효성(Effectiveness) → 실제로 수행 가능해야 함

## 효율성
- 시간 복잡도 (응답시간)

입력 크기 n이 증가할 때 연산 횟수 증가 정도
증가 속도 순서
```
O(1)
→ O(log n)
→ O(n)
→ O(n log n)
→ O(n²)
→ O(n³)
→ O(2ⁿ)
```
- 공간 복잡도 (메모리)

얼마나 많은 메모리를 쓰는가


공간복잡도 줄이는 방법
- 배열 크기 줄이기
- 2차원 → 1차원 압축
- 필요 없는 변수 제거
- 반복문으로 바꿔 재귀 제거

플로이드 워셜 -> 3중 반복문 → O(n³)\
DP -> 일반적으로 O(n²)

| 문제        | 복잡도   |
| --------- | ----- |
| 피보나치      | O(n)  |
| 배낭 (n, k) | O(nk) |
| 비트마스크 DP  | O(2ⁿ) |
