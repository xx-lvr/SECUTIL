## IaaS (Infrastructure as a Service)
**IaaS는 Infrastructure as a Service**로, **서비스형 인프라**를 의미한다.

- IaaS 서비스 제공 업체(CSP)는 인터넷을 통해 인프라 자원을 제공한다.
  - 인프라 자원: 네트워크, 스토리지, 서버, 가상화 환경
- 사용자는 클라우드 콘솔 또는 API를 통해 인프라를 제어하며,
  운영체제(OS), 미들웨어, 런타임, 애플리케이션 및 데이터를 직접 관리한다.
- **관련 서비스:** AWS EC2, NHN Cloud Instance

---

## PaaS (Platform as a Service)
**PaaS는 Platform as a Service**로, **서비스형 플랫폼**을 의미한다.

- IaaS에서 제공하는 인프라에 더해 운영체제(OS)와 미들웨어까지 제공한다.
  - 미들웨어: 운영 체제와 애플리케이션 사이에서 공통 기능을 제공하는 소프트웨어  
    (예: 웹 애플리케이션 서버, 트랜잭션 처리, 데이터 관리)
- 사용자는 애플리케이션 개발, 배포 및 서비스 운영에 집중할 수 있다.
- **관련 서비스:** AWS Elastic Beanstalk

---

## SaaS (Software as a Service)
**SaaS는 Software as a Service**로, **서비스형 소프트웨어**를 의미한다.

- CSP가 인프라(IaaS), 플랫폼(PaaS), 애플리케이션까지 모두 제공한다.
- 사용자는 별도의 설치 없이 웹 브라우저 또는 애플리케이션을 통해 서비스를 이용한다.
- **관련 서비스:** NHN Cloud Dooray, Google Docs, Microsoft 365
