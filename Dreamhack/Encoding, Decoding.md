## 인코딩(Encoding)
```인코딩 = 코드화 = 암호화 = 부호화```\
사람이 인지하지 못하고 컴퓨터가 이해할 수 있도록 바꿔주는 것을 의미한다.

쉽게 **문자(문자열)을 바이트형식으로 변환**한다고 생각하면 편하다.
+ 예시 : ASCII, URL 인코딩, HTML 인코딩, Base64 인코딩

+ 웹브라우저 ----인코딩('멋쟁이' ---->'%EB%A9%8B%EC%9F%81%EC%9D%B4')--------->웹서버
## 디코딩(Decoding)
```디코딩 = 역코드화 = 복호화```\
바이트형식을 문자(문자열)로 변환하는걸 의미한다.
+ 디코딩
웹서버 ---------디코딩('%EB%A9%8B%EC%9F%81%EC%9D%B4' ---->'멋쟁이')---->웹브라우저
